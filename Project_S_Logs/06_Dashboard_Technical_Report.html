01Executive Summary

A concise overview of what was built, why, and where it stands.

Project S is a unified, self-hosted operating system for home servers. The dashboard is its primary interface — a single-page application where users manage all integrated services, monitor system health, and access their self-hosted applications.

Over four rapid iterations on March 23, 2026, the dashboard evolved from a basic card layout to a polished, Apple-inspired glassmorphism interface with scroll-driven animations, 11 switchable color themes, a Recharts-powered monitoring suite, and a Heimdall-style application launcher.

Key Architecture Decisions

  • Tailwind CSS 4 — no config file needed. All theming through CSS custom properties on :root.
  • shadcn/ui — components are copied into the project, not imported from a package. Full ownership, no version lock-in.
  • Recharts over Chart.js — declarative React components, composable chart types, built-in responsive containers, gradient fill support.
  • No state library — React Context is sufficient for theme state. System metrics will use WebSocket subscriptions (future).
  • Single-page scroll model — no router needed for the prototype. Welcome section and dashboard section share one page with scroll-driven transitions.

08Iteration History

Four versions in one day — from scaffolding to polished prototype.

v1 — March 23, 2026
Initial Scaffolding
Basic Next.js setup with module cards in a grid layout. Simple dark background. Straightforward component structure with a top navigation bar and card-based service display.
v2 — March 23, 2026
Client Component Fixes
Fixed React Server Component / Client Component boundary issues. Refactored icon passing from pre-rendered JSX to component references (LucideIcon type). Added "use client" directive to page component.
v3 — March 23, 2026
Apple-Inspired Redesign
Complete visual overhaul. Introduced glassmorphism aesthetic, aurora background with 3 animated gradient blobs, floating icon-only sidebar, 11 color themes with live switching, and proper dark/light mode toggle. Inspired by Apple.com, Obelisk dashboard, and attached reference screenshots.
v4 — March 23, 2026 (Current)
Full Dashboard + Welcome Page
Added scroll-driven two-section layout: Welcome section with 16 application shortcuts and 6 bookmark categories (Heimdall-inspired), fading into a full Recharts monitoring dashboard with CPU/memory gauges, network area charts, system load bars, disk space indicators, alerts, and service status grid. Fixed hydration issues with mounted state pattern. Theme selector popup matches reference design.
