## 📦 What's New in v1.0

### Security Hardening (PR #219)

**Problem:** `boom.sh` had YAML injection risks and silent `.env` creation.

**Solution:**
- ✅ `envsubst` for safe YAML templating
- ✅ YAML validation with PyYAML
- ✅ `.env` creation requires user confirmation
- ✅ Variable renaming to avoid GitGuardian false positives

**Files Changed:**
- `boom.sh` — Safe templating + confirmation prompt
- `config/matrix/homeserver.yaml.tpl` — Converted to envsubst template

---

### Cross-Platform Host Metrics (PR #220)

**Problem:** Dashboard only showed Docker container stats, not actual host metrics.

**Solution:** 3-tier fallback architecture

```
Tier 1: Host Agent (macOS/Windows) — node scripts/host-agent.js
Tier 2: Linux Mounts — /proc, /sys mounted into container
Tier 3: Docker Fallback — container stats only
```

**New Files:**
- `scripts/host-agent.js` — 280 lines, zero dependencies
- Dashboard API — 3-tier fallback logic
- Metrics UI — Platform badge + Agent Connected indicator

## 🚀 Installation

### Prerequisites

- Docker 24+ with Docker Compose
- Node.js 20+ (for host agent on macOS/Windows)
- 10GB free disk space minimum
- 8GB RAM recommended

### Quick Start

```bash
# Clone repo
git clone https://github.com/BasilSuhail/ProjectS-HomeForge.git
cd ProjectS-HomeForge

# Run installer
./boom.sh

# Access dashboard
open http://localhost:3069
```

### First-Time Setup

1. Open http://localhost:3069
2. Generate entropy key (one-time)
3. Create admin account
4. (Optional) Configure Tailscale for remote access
5. (Optional) Run host agent for macOS/Windows metrics

```bash
# Host agent (macOS/Windows only)
node scripts/host-agent.js
```
